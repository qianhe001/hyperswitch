## 卡支付参数
*最近更新时间：2025.07.29*   <!--每次更新建议更新一下日期-->

### 说明1
1. 直连交易接口，请参见[payment](api_jk_zl)，并在公共参数的基础上补充特有参数，以完成交易。
2. 以**paymentOptions.card.authorizationType**字段值，区分预授权（PRE_AUTH）和消费（FINAL_AUTH）。
