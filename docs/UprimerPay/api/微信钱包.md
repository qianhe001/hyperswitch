## 微信支付参数

*最近更新时间：2025.02.20*

### 说明

直连交易接口，请参见[payment](api_jk_zl)，并在公共参数的基础上补充特有参数，以完成交易

### 特有请求参数

| 参数  | 中文名 | 定义  | 长度  | 必填  | 说明  |
| --- | --- | --- | --- | --- | --- |
| paymentMethod | 支付参数 | JSON | <p></p> | M   | paymentMethod |
| products | 商品列表 | JSONArray | <p></p> | M   | 至少填写一个商品 |

| 参数  | 中文名 | 定义  | 长度  | 必填  | 说明  |
| --- | --- | --- | --- | --- | --- |
| type | 商品大类 | String | 32  | O   | 枚举值：  <br>PHYSICAL_GOOD 实物商品  <br>INTANGIBLE_GOOD 虚拟商品  <br>SERVICE 服务  <br>SHIPPING 货运 |
| code | 商品代码 | String | 128 | O   | <p></p> |
| name | 商品名称 | String | 255 | M   | 完整的商品名称 |
| desc | 商品描述 | String | 500 | O   | <p></p> |
| sku | 商品型号 | String | 128 | M   | <p></p> |
| quantity | 商品数量 | Long | 10  | M   | <p></p> |
| totalAmount | 总金额 | Long | 10  | O   | <p></p> |
| url | 商品链接 | String | 255 | O   | <p></p> |

| 参数  | 中文名 | 定义  | 长度  | 必填  | 说明  |
| --- | --- | --- | --- | --- | --- |
| methodType | 支付类型 | String | 32  | M   | 固定枚举值  <br>WECHATPAY - 微信支付 |
| wechatpay | 微信支付参数 | JSON | <p></p> | M   | 微信支付附属参数 |

| 参数  | 中文名 | 定义  | 长度  | 必填  | 说明  |
| --- | --- | --- | --- | --- | --- |
| acceptance | 接口受理方式 | String | 16  | M   | 枚举值：  <br>WEBQR： 扫码支付；  <br>JSAPI： 公众号；  <br>MINIPROG： 小程序；  <br>INAPP： APP支付  <br>MWEB： H5支付 |
| openId | 用户微信公众号ID | String | 64  | C   | 当接口受理方式acceptance=JSAPI公众号或MINIPROG小程序时必填 |
| <font color="rgb(152, 118, 170)">payerIp</font> | <font color="rgb(106, 135, 89)">用户终端IP</font> | String | <font color="rgb(104, 151, 187)">32</font> | C   | H5支付时必填 |

### 特有返回参数

| 参数  | 中文名 | 定义  | 长度  | 非空  | 说明  |
| --- | --- | --- | --- | --- | --- |
| nextAction | 下一步处理 | JSON | <p></p> | O   | 当交易状态返回REQUEST_CUSTOMER_ACTION 时，需要引导客户执行下一步处理 |

| 参数  | 中文名 | 定义  | 长度  | 必填  | 说明  |
| --- | --- | --- | --- | --- | --- |
| actionType | 类型  | String | 32  | M   | 固定枚举值：REDIRECT - 浏览器跳转 |
| url | 跳转地址 | String | 256 | O   | 当actionType为REDIRECT时，需要将客户浏览器跳转到收银台页面 |
| method | HTTP请求方式 | String | 12  | O   | 枚举值：GET、POST |

### 异步通知参数

异步通知参数同同步返回的业务参数
