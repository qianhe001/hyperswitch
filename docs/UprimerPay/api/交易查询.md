## 交易查询
*最近更新时间：2024.11.08*   <!--每次更新建议更新一下日期-->

### 应用场景
商户无需开发收银台，直接调用此接口使用汇付的收银台完成交易。

### 接口说明
* 请求URL：https://acquire.uprimer.com/api/acquire/payment/{id}/get
* 测试环境： https://uatacquire.cloudpnr.com/api/acquire/payment/{id}/get
* 请求方式：GET
* 支持格式：JSON
* id：汇付交易ID或汇付退款交易ID
