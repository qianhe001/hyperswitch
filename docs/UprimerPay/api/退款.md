## 交易退款
*最近更新时间：2024.11.08*   <!--每次更新建议更新一下日期-->

### 应用场景
商户通过通过API 发起退款。

### 接口说明
* 请求URL：https://acquire.uprimer.com/api/acquire/payment/{originalId}/refund
* 测试环境： https://uatacquire.cloudpnr.com/api/acquire/payment/{originalId}/refund
* 请求方式：POST
* 支持格式：JSON
* originalId：原交易的汇付交易ID
